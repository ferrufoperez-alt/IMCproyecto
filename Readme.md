# IMC Calculator (Java Servlet)

Proyecto sencillo de cálculo de IMC desarrollado con **Java Servlets** y desplegado en **Apache Tomcat**.

## Tecnologías usadas

- Java
- Jakarta Servlet API
- Apache Tomcat
- HTML

## Funcionamiento

El usuario introduce:

- Peso
- Altura

El servlet calcula:

IMC = peso / (altura²)

y devuelve el resultado en el navegador.
